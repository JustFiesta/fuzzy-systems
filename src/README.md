# Fuzzy Systems – Cruise Control (Logika Rozmyta)

## Opis projektu
Projekt przedstawia symulację pojazdu poruszającego się po torze, w której utrzymanie zadanej prędkości realizowane jest za pomocą logiki rozmytej (Fuzzy Logic). System działa na zasadzie tempomatu (cruise control), inteligentnie regulując moc silnika w oparciu o bieżące zachowanie pojazdu.

Zamiast stosować sztywne progi i klasyczne sterowanie binarne, kontroler rozmyty wykorzystuje pojęcia lingwistyczne, które odwzorowują ludzkie decyzje, takie jak:
- „za szybko”,
- „za wolno”,
- „przyspiesza”,
- „bez zmian”.

Dzięki temu pojazd reaguje płynnie i stabilnie, minimalizując nagłe zmiany prędkości.

## Cel projektu
Głównym celem jest demonstracja zastosowania logiki rozmytej w systemie sterowania prędkością pojazdu oraz analiza jej wpływu na komfort i stabilność jazdy.

Projekt ma charakter edukacyjno-badawczy i stanowi podstawę do dalszej rozbudowy, np. o interakcję wielu pojazdów.

## Główne założenia
- Sterowanie prędkością realizowane wyłącznie przez kontroler rozmyty
- Uproszczony model fizyczny pojazdu
- Symulacja czasu rzeczywistego
- Modularna architektura
- Zasady:
  - **KISS (Keep It Simple, Stupid)**
  - **Single Responsibility Principle**
